# Flash-Hub (优选社交点评系统)

这是一个基于 Spring Boot 2.x 的高性能社交后端系统。项目核心攻克了高并发场景下的秒杀安全、多级缓存一致性以及社交 Feed 流分发等技术难点。

## 🌟 技术栈
- **核心框架**: Spring Boot, MyBatis Plus
- **数据存储**: MySQL, Redis (StringRedisTemplate, Redisson)
- **核心工具**: Hutool, Lombok

## 🚀 核心功能与技术实现

### 1. 高并发秒杀模块 (分布式锁 & 原子性)
- **原子性操作**: 针对优惠券抢购，编写 **Lua 脚本** 封装“库存校验-一人一单判断-扣减库存”逻辑，确保在高并发下的原子执行。
- **分布式锁**: 集成 **Redisson** 框架，解决集群环境下 JDK 原生锁失效问题，并实现看门狗（Watch Dog）自动续期与锁重入机制。
- **性能优化**: 采用 **CAS 乐观锁** 策略防御超卖问题，在不引入重型消息队列的前提下保证了万级 QPS 的吞吐量。

### 2. 企业级缓存实战 (高性能查询)
- **缓存击穿解决方案**: 实现 **逻辑过期 (Logical Expire)** 机制，配合自定义线程池实现缓存异步重建，保障热点数据查询的零延迟响应。
- **缓存穿透防护**: 采用缓存空对象策略，防御无效请求对数据库的直接冲击。

### 3. 社交 Feed 流 (推拉模式研究)
- **高效分发**: 实现**推模式 (Push)** 的关注博文分发，利用 Redis **ZSet** 数据结构按照时间戳排序，构建高性能用户收件箱。
- **共同关注**: 基于 Redis **Set** 集合的交集运算实现好友关联推荐。

## 📂 项目结构亮点
- `src/main/resources/*.lua`: 核心业务的原子性保障脚本。
- `src/main/java/com/hmdp/service/impl`: 采用 AOP 代理模式处理事务，解决同类方法调用导致的事务失效问题。
